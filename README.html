What’s it?

A try to migrate RSJUG site to Jekyll and github pages.

Features

  • Responsive layout

  • Mobile 'friendly'

  • Disqus integration

  • Based on github pages

  • Automatically build on git push

  • Posts based on Asciidoc (via asciidoctor.org/[Asciidoctor] and Markdown (focus on content instead of markup)

Running it locally

  1. Fist you will need to install Ruby

    Use Ruby 2.1 or superior

  2. Install Bundler gem:

    gem install bundler
    You can skip this step if you want to install gems individually, see requied gems here

  3. Clone the project:

    git clone https://github.com/rmpestano/site rsjug-site
cd rsjug-site

  4. Run

    bundle install
    It will install gems declared in Gemfile. You can skip this step if gems are already installed

  5. Then run Jekyll

    jekyll serve

  6. And finally access http://localhost:4000/site

Every file change is automatically published locally by Jekyll.
